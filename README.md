# 🛡️ lazyguard

**lazyguard** — это легковесная утилита на Go для активной защиты серверов от DoS/DDoS атак (включая HTTP-флуд, TCP/UDP-флуд и атаки на VLESS/Xray порты). 

Она работает независимо от используемого веб-сервера или VPN-протокола, анализируя сетевой трафик напрямую через `libpcap` и автоматически блокируя атакующие IP-адреса на уровне ядра с помощью `iptables`.

## ✨ Особенности
* **Полная независимость:** Работает с Nginx, Apache, Xray, Sing-box, Docker и любыми другими сервисами.
* **Высокая производительность:** Написана на Go с использованием CGO для низкоуровневого анализа пакетов.
* **Умная блокировка:** Разделяет активность на подозрительную (логирует в фоне) и критическую (отправляет в бан).
* **Автоматический разбан:** Сама снимает блокировку с IP по истечении заданного времени.

## 🚀 Быстрый старт (Установка и сборка)

Для того чтобы склонировать репозиторий, установить все системные зависимости (компилятор, библиотеки захвата пакетов) и собрать оптимизированный бинарник, выполните **одну команду** в терминале вашего сервера:

```bash
git clone [https://github.com/internetkafe/lazyguard.git](https://github.com/internetkafe/lazyguard.git) && cd lazyguard && sudo apt-get update && sudo apt-get install -y build-essential libpcap-dev iptables && go mod tidy && CGO_ENABLED=1 go build -o lazyguard main.go
